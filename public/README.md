# Public Assets Folder

This folder contains static assets that will be served by Next.js.

## Logo Placement

**Place your PathoCare logo here as `logo.png`**

- Supported formats: PNG, SVG, JPG, WEBP
- Recommended: PNG with transparent background
- Recommended size: 512x512 pixels or larger for high-resolution displays
- The logo will be accessible at `/logo.png` in your website

## Other Assets

You can also place other static assets here:
- Images: `/images/your-image.jpg`
- Icons: `/icons/icon.svg`
- Fonts: `/fonts/your-font.woff2`
- Documents: `/documents/brochure.pdf`

All files in this folder are publicly accessible from the root URL path.
